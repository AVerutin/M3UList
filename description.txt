Общие положения системы управления списками воспроизведения:

1. Все каналы хранятся в локадьной базе данных.
2. При запуске приложения отображаются в табличном виде все каналы,
   имеюшиеся в базе данных.
3. При импорте каналов из текстового файла списка воспроизведения,
   производится парсинг и добавление считанных каналов в базу данных.
4. При формировании нового списка вопроизведения заполняются параметры
   списка (наименования и пр.), а также добавляются выбранные каналы,
   имеющиеся в базе данных.
5. При формировании списка предусмотрена возможность группировки каналов
   по категориям, список которых также хранится в локальной базе данных.
6. После формирования списка воспроизведения необходим экспорт его в
   текстовый формат списка воспроизведения.
7. Для управления группами каналов и звуковых дорожек предусмотрены
   соответствующие формы.

   На главной форме отображено главное меню приложения и табличное представление списка каналов,
с возможностью выделения нескольких строк, удаления из базы данных несколько выделенных каналов,
редактирования одного выбранного канала (если выбрано несколько строк, то последнего выделенного)


   Главное меню включает:
1. "Файл", содержащий пункты - "Импорт каналов из списка", "Экспорт каналов
   в список", "Выход из приложения".
2. "Каналы", содержащий пункты - "Добавить канал", "Удалить канал", "Редактировать канал",
   "Добавить в список".

Возможна замена главного меню приложения на управляющие кнопки на главной форме приложения.
